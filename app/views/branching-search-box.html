Branching from a search box

Branching from a search box within a coded prototype design enables you to significantly enhance the realism, flexibility, and testability of your prototype

Branching allows the prototype to respond differently based on user input. By enabling a user to be able to type in multiple search terms and end up at different pages mimics how a real system would behave. This in turn helps to make usability testing more meaningful.
