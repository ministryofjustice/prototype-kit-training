Populate tables through looping JSON data

To loop data through a json folder is a handy technique of saving time when prototyping if encountering a situation where you have to produce a lot of tables in a case management system.

For this example the looping of data will be used to populate a table. This method helps with scalability and flexibility by allowing the adding or modifying of data without changing the overall structure of your code. It also will help in terms of iterating of designs, speeding up the time it takes to test different data sets.

First we will need to have a page set up that has a table that we can loop the data into.

Once the page is set up with the table in place we need to go about setting up the json parts.

Create a folder in the data section of your prototype and save it .json , so for example tables.json
