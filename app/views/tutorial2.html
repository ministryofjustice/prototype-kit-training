    • Tutorial Two: Passing data between pages

    To begin passing data between pages we need to modify our question and check answers pages.

    On the first question page, take note of the input and more specifically the name. For example the one used in this tutorial is name="currentname" . This is what we shall use on the check answers page to then get the prototype to resurface whatever is typed into that input

    Within the check your answers page locate the value and text for the first question. In the text section enter the following data['currentname'] or ['Current name not entered'] . The video before will help you make sure this is done correctly.

    It should look like the above video for question one.

    For passing data from the second radio button question we need to again remember the value for the radio buttons. Go to the code for this page and our example is name="choicenewname" .

    Also on this page you will need to change the values on each of the options. In our example we have a yes or no radio button choice so change the text to value="Yes" and value="No" . This will mean that whichever choice is selected the right value will be displayed on the check answers page.

    Now back on the check answers page we need to again get the prototype to resurface this remembered data in the right place. Repeat the same step of entering the right pieces of code in the correlating place for the second question. Place the following data['choicenewname'] or ['Current name not entered'] . This will again bring through and display whatever choice is made on the radio buttons.

    It should look like the above video for question two.

    The final step to this is just completing these steps for the last question

    Repeat the same step of changing the code for the name in the input. For the example we have used it will be as follows name="newname"

    Finally place the data['newname'] or ['Current name not entered'] in the text section for the third question on the check answers page.

    It should look like the above video for question three.
