    Getting started - Code editors

    Before we can get started with creating prototypes, we first need to check we have a few things downloaded and set up.

    Firstly, you will need to download a code editor, which are available from self service in the developer tools section. The names of the most popular ones are Visual Code Studio, Sublime and Atom

    Visual Studio Code, also referred to as VS Code, is the preferred code editor due to its continued support. It works on Windows, Linux and macOS.
